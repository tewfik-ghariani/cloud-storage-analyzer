The results of the query are shown in the grid.
You may perform several actions such as:
  • Filter a single column using:
    • Equals
    • Not Equal
    • Starts with
    • Ends with
    • Contains
    • Not contains
  • Search the whole grid
  • Resize the columns
  • Sort the columns (ascending/descending order)
  • Move the columns
  • Export as CSV
